# Mixpanel 연동 테스트 체크리스트

## ✅ 현재 상태 확인

### 1. 토큰 설정 확인
- ✅ 토큰: `951a42e2009f89c11e01ea359dac0810`
- ✅ `mixpanel-config.js` 파일에 저장됨

### 2. 웹사이트에서 테스트하기

#### Step 1: 웹사이트 접속
- `http://localhost:3000` 또는 배포된 URL로 접속
- 또는 `index.html` 파일을 브라우저에서 열기

#### Step 2: 브라우저 콘솔 확인
1. F12로 개발자 도구 열기
2. Console 탭 확인
3. 다음 메시지가 보여야 함:
   - `✅ Mixpanel 초기화 완료`

#### Step 3: 이벤트 생성하기
웹사이트에서 다음 활동을 수행하세요:

1. **페이지 방문**
   - 홈페이지 → 제품 페이지 → 회사소개 → 문의하기
   - 각 페이지 방문 시 "Page Viewed" 이벤트 생성

2. **버튼 클릭**
   - "제품 둘러보기" 버튼 클릭
   - "견적 문의" 버튼 클릭
   - "무료 상담 신청" 버튼 클릭
   - 각 클릭 시 "Button Clicked" 이벤트 생성

3. **제품 필터 사용** (제품 페이지에서)
   - "메모리", "시스템", "아날로그", "RF" 필터 버튼 클릭
   - 각 클릭 시 "Product Filtered" 이벤트 생성

4. **제품 견적 요청** (제품 페이지에서)
   - 제품 카드의 "견적 요청" 버튼 클릭
   - "Product Quote Requested" 이벤트 생성

5. **폼 제출** (문의하기 페이지에서)
   - 폼 작성 후 제출
   - "Form Submitted" 이벤트 생성

#### Step 4: Mixpanel 대시보드에서 확인
1. Mixpanel 대시보드 접속
2. 좌측 메뉴에서 **"Data"** 클릭
3. **"Events"** 클릭
4. 이벤트 목록 확인
5. 1-2분 후 이벤트가 표시됩니다

## 🔍 문제 해결

### 이벤트가 보이지 않는 경우

1. **브라우저 콘솔 확인**
   - "✅ Mixpanel 초기화 완료" 메시지가 있는지 확인
   - 오류 메시지가 있는지 확인

2. **네트워크 탭 확인**
   - DevTools의 Network 탭 열기
   - "mixpanel" 또는 "api.mixpanel.com" 요청이 있는지 확인
   - 요청이 200 OK로 성공했는지 확인

3. **시간 기다리기**
   - Mixpanel에 이벤트가 표시되기까지 몇 분 걸릴 수 있습니다
   - 5-10분 후 다시 확인해보세요

4. **토큰 재확인**
   - `mixpanel-config.js` 파일에서 토큰이 올바르게 설정되었는지 확인
   - 따옴표 안에 토큰이 있는지 확인

## 📊 확인할 이벤트 목록

Mixpanel 대시보드에서 다음 이벤트들을 확인할 수 있어야 합니다:

- ✅ **Page Viewed** - 페이지 방문
- ✅ **Button Clicked** - 버튼 클릭
- ✅ **Product Filtered** - 제품 필터 사용
- ✅ **Product Quote Requested** - 제품 견적 요청
- ✅ **Form Submitted** - 폼 제출

- 이벤트는 실시간이 아닐 수 있습니다 (몇 분 지연 가능)

- 다른 브라우저에서도 테스트해보세요
