# Google Sheets API 활성화 가이드

## 오류 메시지
"Google Sheets API has not been used in project 1009573505890 before or it is disabled"

## 해결 방법

### 1단계: Google Sheets API 활성화
브라우저에서 다음 링크로 이동:
https://console.developers.google.com/apis/api/sheets.googleapis.com/overview?project=1009573505890

### 2단계: API 사용 설정
1. 페이지가 열리면 **"사용 설정"** 또는 **"Enable"** 버튼 클릭
2. 잠시 기다리면 API가 활성화됩니다

### 3단계: Google Drive API도 활성화 (필요한 경우)
스프레드시트 생성 기능을 사용하려면 Google Drive API도 필요할 수 있습니다:
https://console.developers.google.com/apis/api/drive.googleapis.com/overview?project=1009573505890

### 4단계: 확인
API 활성화 후:
- 몇 분 정도 기다려주세요 (전파 시간)
- 다시 폼을 제출해보세요
