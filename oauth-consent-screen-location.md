# OAuth 동의 화면 위치 가이드

## 현재 위치
현재 **OAuth 개요** 페이지에 있습니다.

## OAuth 동의 화면으로 이동하는 방법

### 방법 1: 좌측 사이드바 사용
1. 좌측 사이드바에서 **"대상"** (Target) 메뉴 클릭
2. 또는 **"설정"** (Settings) 메뉴 클릭
3. 여기서 OAuth 동의 화면 설정을 찾을 수 있습니다

### 방법 2: 직접 URL 접속
다음 URL로 직접 이동:
```
https://console.cloud.google.com/apis/credentials/consent
```

### 방법 3: 메뉴 경로
1. **API 및 서비스** (왼쪽 상단 햄버거 메뉴)
2. **OAuth 동의 화면** 클릭

## OAuth 동의 화면에서 해야 할 일

1. **테스트 사용자** 섹션 찾기
2. **+ 사용자 추가** 버튼 클릭
3. 이메일 주소 입력: `joliekjh@gmail.com`
4. **추가** 클릭
5. **저장** 클릭

## 빠른 링크
