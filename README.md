# TechSemiconductor - 반도체 B2B 웹사이트

반도체 B2B 서비스를 위한 현대적이고 전문적인 웹사이트입니다.

## 주요 기능

- **홈페이지**: 회사 소개, 주요 특징, 제품 카테고리, 통계 정보
- **제품 페이지**: 반도체 제품 카탈로그 (메모리, 시스템, 아날로그, RF)
- **회사소개 페이지**: 회사 미션, 핵심 가치, 연혁, 리더십 팀, 인증 정보
- **문의하기 페이지**: 연락처 정보 및 문의 양식

## 기술 스택

- HTML5
- CSS3 (반응형 디자인)
- JavaScript (바닐라 JS)
- Google Fonts (Noto Sans KR)

## 파일 구조

```
.
├── index.html          # 메인 홈페이지
├── products.html       # 제품 카탈로그 페이지
├── about.html          # 회사소개 페이지
├── contact.html        # 문의하기 페이지
├── styles.css          # 스타일시트
├── script.js           # JavaScript 기능
└── README.md           # 프로젝트 설명서
```

## 시작하기

### 로컬에서 실행하기

1. 모든 파일을 다운로드하거나 클론합니다.

2. 웹 서버를 실행합니다. 예를 들어:
   - Python 3: `python -m http.server 8000`
   - Node.js (http-server): `npx http-server`
   - VS Code Live Server 확장 프로그램 사용

3. 브라우저에서 `http://localhost:8000` (또는 사용한 포트)로 접속합니다.

### 직접 열기

파일을 직접 브라우저에서 열 수도 있지만, 일부 기능은 웹 서버가 필요할 수 있습니다.

## 주요 특징

### 반응형 디자인
- 모바일, 태블릿, 데스크톱 모든 기기에서 최적화된 레이아웃
- 햄버거 메뉴를 통한 모바일 네비게이션

### 인터랙티브 기능
- 제품 필터링 기능
- 스크롤 애니메이션
- 통계 숫자 카운터 애니메이션
- 문의 양식 처리

### 디자인 요소
- 현대적인 그라데이션 배경
- 부드러운 호버 효과
- 카드 기반 레이아웃
- 일관된 색상 체계

## 커스터마이징

### 색상 변경
`styles.css` 파일의 `:root` 변수를 수정하여 색상을 변경할 수 있습니다:

```css
:root {
    --primary-color: #2563eb;
    --primary-dark: #1e40af;
    --secondary-color: #64748b;
    /* ... */
}
```

### 내용 수정
각 HTML 파일을 열어 텍스트, 이미지, 링크 등을 수정할 수 있습니다.

### 기능 추가
`script.js` 파일에 새로운 JavaScript 기능을 추가할 수 있습니다.
