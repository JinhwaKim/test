# Mixpanel에서 이벤트 확인하는 방법

## 📊 이벤트 확인 방법

### 방법 1: Data > Events 메뉴 사용

1. 좌측 사이드바에서 **"Data"** 클릭
2. **"Events"** 클릭
3. 이벤트 목록이 표시됩니다
4. "Page Viewed", "Button Clicked", "Form Submitted" 등의 이벤트를 확인할 수 있습니다

### 방법 2: Home 대시보드에서 확인

1. **"Home"** 페이지에 있는 **"New Events"** 위젯 확인
2. "No events found" 메시지가 보이면 아직 이벤트가 전송되지 않은 것입니다
3. 웹사이트에서 활동을 하면 여기에 표시됩니다

### 방법 3: Live View 사용

1. 좌측 사이드바에서 **"Data"** 클릭
2. **"Events"** 클릭
3. 상단에 **"Live View"** 또는 **"Live"** 탭이 있을 수 있습니다
4. 실시간으로 들어오는 이벤트를 확인할 수 있습니다

## 🔍 이벤트가 보이지 않는 경우

### 1. 웹사이트에서 활동하기
- 페이지를 새로고침하세요
- 버튼을 클릭해보세요
- 제품 필터를 사용해보세요
- 폼을 제출해보세요

### 2. 몇 분 기다리기
- Mixpanel에 이벤트가 전송되고 표시되기까지 몇 분이 걸릴 수 있습니다

### 3. 브라우저 콘솔 확인
- F12를 눌러 개발자 도구 열기
- Console 탭에서 Mixpanel 관련 메시지 확인
- "✅ Mixpanel 초기화 완료" 메시지가 보여야 합니다

### 4. 토큰 확인
- `mixpanel-config.js` 파일에서 토큰이 제대로 설정되었는지 확인
- `YOUR_MIXPANEL_TOKEN_HERE`가 아닌 실제 토큰이어야 합니다

## 📝 확인해야 할 이벤트들

웹사이트에서 다음 활동을 하면 이벤트가 생성됩니다:

1. **Page Viewed** - 페이지 방문 시
2. **Button Clicked** - 버튼 클릭 시
3. **Product Filtered** - 제품 필터 사용 시
4. **Product Quote Requested** - 제품 견적 요청 시
5. **Form Submitted** - 폼 제출 시

## 🧪 테스트 방법

1. 웹사이트 접속
2. 여러 페이지를 방문해보세요
3. 버튼을 클릭해보세요
4. 제품 페이지에서 필터를 사용해보세요
5. 문의 폼을 제출해보세요
6. Mixpanel 대시보드에서 **Data > Events**로 이동하여 확인

## 💡 팁

- 이벤트가 실시간으로 보이지 않으면 5-10분 후 다시 확인해보세요
- 브라우저 캐시를 삭제하고 다시 시도해보세요
- 다른 브라우저에서도 테스트해보세요
